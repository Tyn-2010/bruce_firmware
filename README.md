# Bruce Firmware - Custom Fork

This is a fork of the original Bruce Firmware project.

Original project: [https://github.com/pr3y/Bruce](https://github.com/BruceDevices/firmware)

Original: [README.md](https://github.com/BruceDevices/firmware/blob/main/README.md)

## About this fork

This fork includes custom changes and improvements made for personal use and testing.

## Changes in this fork

- Added support for ESP32-S3 N16R8
- Added 128x160 ST7735 display support
- Added button configuration for custom board
- Fixed TFT define conflicts in build configuration

## How to keep this fork updated

This project is kept in sync with the original Bruce Firmware repository using an upstream remote.
Changes from the original project can be merged regularly while keeping custom modifications.
